# Documentation
- Adopt ADRs.
- Maintain relevant documentation.
- When creating documentation, add reference links in existing docs to the new docs. Always cross reference files.
- Avoid duplicating docs.
- When completing a task, make sure all documentation is up to date.
- When completing a task, create a document that summarizes the task, the prompts and any other related context used to get the task done. Include that doc in the commits.
